# Blender Python 脚本

一个基本的 Blender Python 脚本（使用 bpy 模块）通常遵循这样的结构：导入必要的模块，通过 API 访问 Blender 数据，并为特定任务定义类或函数。而且 Blender 里面的一切都是操作，都是一个 Python 函数。 所以基本上，如果可以用 UI 做到，就应该可以用脚本写出来。

## 核心 API 模组


|关键模块|概念|
|:---:|:---|
|bpy|主要模块，用于提供连接 Python API，对数据、运算符和用户界面的访问|
|bpy.data|提供对当前文件中所有数据的全局访问，例如对象、网格、材质和场景|
|bpy.context|提供对当前状态的访问，例如活动对象、选定对象和活动窗口|
|bpy.op|用于调用的内置运算符，这些运算符是用户界面中可用的工具和操作（例如，添加立方体、应用修改器）|

## 脚本的基本示例

[Blender Python API 文件](https://docs.blender.org/api/current/index.html)

- Blender 注册 Class

```py
import bpy

# 遵循 PEP8 类名风格指南

class SimpleOperator(bpy.types.Operator):
  bl_idname = "object.simple_operator" # 唯一的内部名称（小写，以下划线分隔）
  bl_label = "简单运算符" # 易于理解的名称
  bl_options = {'REGISTER', 'UNDO'} # 撤销等选项

  def execute(self, context): 
    # 运算符的主要逻辑写在这里，此示例仅向控制台打印一条消息并添加一个立方体
    print(f"Hello Blender! Adding a cube at {context.scene.cursor.location}")
    bpy.ops.mesh.primitive_cube_add(location=context.scene.cursor.location)
    return {'FINISHED'} # 告诉 Blender 操作已完成 成功

# --- 注册 --- 此部分在 Blender 中执行脚本时运行
  def register():
    bpy.utils.register_class(SimpleOperator)
    print("SimpleOperator 已注册")

  def unregister():
    bpy.utils.unregister_class(SimpleOperator)
    print("SimpleOperator 已取消注册")

# 允许脚本直接在 Blender 的脚本工作区运行
if __name__ == "__main__":
  register()
```

- Blender Python 脚本向场景中添加一个立方体，可以使用 bpy.ops.mesh.primitive_cube_add() 运算符。此脚本演示了如何在指定位置添加立方体并调整其大小。


```py
import bpy

def add_cube_example():

  # 向当前 Blender 场景添加一个具有特定属性的立方体。
  # 检查用户是否处于正确的模式（对象模式最适合添加对象）
  if bpy.context.mode != 'OBJECT':

    bpy.ops.object.mode_set(mode='OBJECT')

    # 向场景添加一个新的立方体图元
    # 可根据需要调整参数：
    # size: 初始缩放比例（Blender 单位）
    # enter_editmode: False 表示创建后保持在对象模式
    # align: 新对象的对齐方式（世界坐标系、视图坐标系、光标坐标系）
    # location: X、Y、Z 坐标
    # scale: X、Y、Z 缩放比例
    bpy.ops.mesh.primitive_cube_add(size=1, enter_editmode=False, align='WORLD', location=(0, 0, 0), scale=(2, 1, 0.5))

    # 可选：您可以获取新创建对象的引用以对其进行进一步修改
    new_cube = bpy.context.active_object
    new_cube.name = "MyScriptedBox"
    print(f"添加的立方体名称：{new_cube.name}")

if __name__ == "__main__":
  add_cube_example()
```


 - 在 3D 視圖側邊欄（N 面板）顯示「第一個修改器」屬性的簡單腳本範例


```py
import bpy

class TEST_PT_ModifierUI(bpy.types.Panel):
  bl_label = "現有修改器控制"
  bl_idname = "TEST_PT_modifier_ui"
  bl_space_type = 'VIEW_3D'
  bl_region_type = 'UI'
  bl_category = 'MyTab'

  def draw(self, context):
    layout = self.layout
    obj = context.active_object

    if obj and obj.modifiers:
      # 取得第一個修改器作為範例
      mod = obj.modifiers[0]
      layout.label(text=f"修改器: {mod.name} ({mod.type})")
            
      # 根據修改器類型顯示對應屬性
      if mod.type == 'SUBSURF':
        layout.prop(mod, "levels", text="視圖細分")
        layout.prop(mod, "render_levels", text="渲染細分")
      elif mod.type == 'BEVEL':
        layout.prop(mod, "width", text="寬度")
        layout.prop(mod, "segments", text="段數")
      else:
        layout.label(text="未定義此類型的 UI 顯示")
    else:
      layout.label(text="請選擇含有修改器的物體")

def register():
  bpy.utils.register_class(TEST_PT_ModifierUI)

def unregister():
  bpy.utils.unregister_class(TEST_PT_ModifierUI)

if __name__ == "__main__":
  register()
```

 - 根据坐标列表创建曲线的基本脚本

```py
import bpy

# 定义路径点
coords_list = [(0,0,0), (1,1,0), (2,-1,0), (3,0,0)]

# 创建曲线数据和新的样条曲线
crv = bpy.data.curves.new('ProceduralPath', 'CURVE')
crv.dimensions = '3D'
spline = crv.splines.new(type='NURBS')

# 设置点
spline.points.add(len(coords_list) - 1)
for i, coord in enumerate(coords_list):
  spline.points[i].co = (coord[0], coord[1], coord[2], 1)

# 链接到场景
obj = bpy.data.objects.new('PathObject',  crv)

bpy.context.collection.objects.link(obj)
```
